cookiecutter-zweb
=================

A zweb template for `cookiecutter`_.

.. _cookiecutter: https://github.com/audreyr/cookiecutter


Use it now
----------

.. code-block:: text

    $ pip install cookiecutter
    $ cookiecutter https://github.com/hilarryxu/cookiecutter-zweb.git
